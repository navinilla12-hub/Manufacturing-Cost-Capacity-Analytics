# Manufacturing Cost & Capacity Analytics System

## 📌 Overview
This project simulates a real-world manufacturing environment to analyze production efficiency, cost structure, and capacity utilization. It is designed to replicate industry use cases where analysts evaluate operational performance and identify opportunities for optimization.

The system generates realistic plant-level data and applies business analytics techniques to uncover insights related to cost efficiency and production bottlenecks.

---

## 🎯 Problem Statement
Manufacturing organizations need to:
- Understand cost per unit of production
- Monitor capacity utilization across plants
- Identify bottlenecks and inefficiencies
- Optimize production performance and reduce operational costs

This project addresses these challenges using a data-driven approach.

---

## 🧠 Approach

### 1. Data Simulation
- Generated synthetic manufacturing dataset
- Included:
  - Plant-level production data
  - Product categories (Rubber, Polymer, Chemical)
  - Units produced
  - Maximum production capacity
  - Fixed and variable production costs

### 2. Feature Engineering
- Cost per Unit = Production Cost / Units Produced
- Capacity Utilization = Units Produced / Max Capacity

### 3. KPI Analysis
- Average cost per unit
- Average capacity utilization
- Identification of high-cost operations

### 4. Bottleneck Detection
- Identified production units operating below optimal capacity (<60%)
- Highlighted inefficiencies in resource utilization

### 5. Operational Insights
- Compared performance across plants
- Analyzed relationship between utilization and cost efficiency

---

## 📊 Key Insights

- Lower capacity utilization leads to higher cost per unit
- Underutilized plants indicate operational inefficiencies
- High production cost areas highlight optimization opportunities
- Plant-level comparisons help identify best and worst performers
